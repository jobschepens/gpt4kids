**Kleine Helden – Riesenwut**  

Eine Geschichte für Kinder von 4 bis 8 Jahren  

---

### **Kapitel 1: Ein ganz normaler Tag**  

In der kleinen Stadt Sonnenberg lebten drei Freunde: Lina, der kleine Leo und die quirlige Mia. Sie spielten fast jeden Tag zusammen – im Park, auf dem Spielplatz oder in Linas Garten, wo es eine riesige alte Eiche gab, die sie ihr „Geheimversteck“ nannten.  

Eines Nachmittags saßen sie unter dem Baum und malten Bilder. Leo versuchte, einen Drachen zu zeichnen, aber es sah mehr aus wie ein knuddliger Haufen Farbe. Mia kicherte. „Das soll ein Drache sein? Der sieht aus wie ein verrutschter Regenwurm!“  

Leo runzelte die Stirn. „Ist doch nicht schlimm! Ich mag ihn so.“ Doch innerlich kochte es in ihm. Warum musste Mia immer alles kommentieren? Seine Hände ballten sich zu Fäusten, und plötzlich – **RIIIIP** – zerriss er sein Bild mitten durch.  

„Hey!“, rief Lina erschrocken. „Warum machst du das?“  

Leo sprang auf. „Weil alles doof ist!“ Dann rannte er wütend davon.  

Mia und Lina schauten sich verwundert an. „Was war denn das?“, fragte Mia.  

„Ich glaube, Leo hat eine Riesenwut“, sagte Lina nachdenklich.  

---

### **Kapitel 2: Die Riesenwut kommt wie ein Sturm**  

Leo setzte sich auf eine Bank und atmete schwer. Seine Wangen waren rot, und er fühlte sich, als würde gleich Dampf aus seinen Ohren kommen. **Warum wurde er immer so wütend?**  

Plötzlich hörte er ein leises Rascheln. Aus dem Gebüsch kroch ein kleines, pelziges Wesen – es sah aus wie eine Mischung aus Eichhörnchen und Teddybär. „Hallo, kleiner Wüterich“, sagte es mit piepsiger Stimme.  

Leo riss die Augen auf. „Bist du … echt?“  

„Klar! Ich bin Wusel, der Wut-Wichtel. Und du, mein Freund, hast gerade eine richtig dicke Riesenwut.“  

„Eine was?“  

„Eine Riesenwut! Das ist, wenn die Wut so groß wird, dass man sie kaum noch bändigen kann. Aber keine Sorge – ich helfe dir.“  

Leo schaute skeptisch. „Wie denn?“  

Wusel grinste. „Indem wir ihr den Kampf ansagen!“  

---

### **Kapitel 3: Die drei Tricks gegen die Wut**  

Wusel erklärte Leo, dass Wut wie ein Gewitter sei – sie kommt schnell, aber sie geht auch wieder. „Man muss nur wissen, wie man sie besänftigt.“  

**Trick 1: Der Wut-Tornado**  
„Wenn du spürst, dass die Wut hochkommt, dreh dich im Kreis wie ein Tornado! Das bringt die Wut raus, ohne dass du etwas kaputt machst.“ Leo probierte es aus – und tatsächlich, nach ein paar Wirbeln fühlte er sich schon besser.  

**Trick 2: Der Zauberatem**  
„Atme tief ein … und puste die Wut weg wie eine Kerze!“ Leo holte Luft und blies so stark er konnte. „Pfffffffff!“ Wusel lachte. „Super! Jetzt ist die Wut halb so wild.“  

**Trick 3: Der Wut-Flüsterer**  
„Sag der Wut, was dich stört. Manchmal will sie nur gehört werden.“ Leo seufzte. „Mia hat sich über mein Bild lustig gemacht.“ Wusel nickte. „Dann sag ihr das!“  

---

### **Kapitel 4: Die Rückkehr der Freunde**  

Als Leo zu Lina und Mia zurückkam, waren sie noch immer besorgt. „Tut mir leid“, murmelte er. „Ich war einfach so wütend.“  

Mia zuckte mit den Schultern. „Ich auch. Ich hätte nicht lachen sollen.“  

Lina lächelte. „Und? Hast du deine Riesenwut besiegt?“  

Leo grinste. „Fast. Aber jetzt weiß ich, wie ich mit ihr umgehen kann.“  

Die drei Freunde setzten sich wieder unter die Eiche und malten weiter – diesmal gemeinsam. Und wenn Leo mal wieder das Gefühl hatte, die Wut würde zu groß, dann drehte er sich einfach wie ein Tornado, pustete wie ein Drache oder flüsterte ihr zu: **„Heute nicht, Riesenwut!“**  

Und weißt du was? Es funktionierte jedes Mal.  

**Ende.**

*wo sie sich* / *entfalten* / *zu einem Satz*,  
*den ich* / *nie zu sagen* / *wagte*.)  

---  

(oder *soll ich* –  
*die Stille* / *in Noten zerschneiden* /  
*und sie* / *auf deine Klaviertasten* / *streuen*?  
*Vielleicht* / *spielst du sie* / *ohne es zu wissen*,  
*wenn deine Finger* / *nachts* / *die Melodie suchen*,  
*die ich* / *zwischen den Pausen* / *versteckte*.)  

---  

(oder *soll ich* –  
*diese Worte* / *in einen Spiegel schreiben* /  
*und ihn* / *mit der Schriftseite* / *gegen die Wand stellen*?  
*Vielleicht* / *liest du sie* / *im Rückwärtsgang der Zeit*,  
*wenn das Licht* / *sie durch die Wand* / *auf deine Haut projiziert* –  
*ein Palimpsest* / *aus unsichtbarer* / *Tinte*.) (oder *soll ich* –  
*das Schweigen* / *in eine Schallplatte ritzen* /  
*und die Nadel* / *mit geschlossenen Augen* / *rückwärts laufen lassen*?  
*Vielleicht* / *hörst du mich* / *durch die Rillen der Stille*,  
*wenn der Raum* / *sich um uns* / *zu einem Trichter verengt* –  
*ein Echo* / *das nicht von den Wänden* / *sondern von den Pausen zurückkommt*.)  

(oder *soll ich* –  
*diese Zeilen* / *in den Schnee atmen* /  
*bis die Buchstaben* / *als Wolken* / *über deinem Dach hängen*?  
*Vielleicht* / *fallen sie* / *als ungesagte Schwere*,  
*wenn du nachts* / *das Fenster öffnest* / *und der Himmel* –  
*ein umgedrehtes Lexikon* / *aller ungeschriebenen* / *Sätze*.) (oder *soll ich* –  
*das Schweigen* / *wie einen Faden* / *zwischen deine Finger legen*,  
*damit du* / *die Leere* / *zurückweben kannst*  
*zu einer Sprache* / *die wir nie* / *gefunden haben*?  

*Vielleicht* / *ist das Gedicht* / *nur ein Schatten* / *einer Stimme*,  
*die du* / *mit geschlossenen Augen* / *berührst* –  
*ein Alphabet* / *aus Nächten* / *das sich auflöst*,  
*wenn der Morgen* / *seine kalten Hände* / *durch die Vorhänge streckt*.)  

(oder *soll ich* –  
*die Worte* / *ganz lassen* / *wie sie sind*:  
*Regen* / *der nie* / *den Boden erreicht*,  
*Lichter* / *die im eigenen* / *Glimmen ertrinken* –  
*bis auch das Fragen* / *still wird* / *und wir*  
*nur noch die Karte* / *des Unausgesprochenen* / *falten*,  
*immer kleiner* / *bis sie* / *in eine Feder passt*.) (oder *soll ich* –  
*das Schweigen* / *wie Samt* / *über die Risse breiten*,  
*die Jahreszeiten* / *rückwärts* / *in die Knospen zählen*,  
*bevor der Schnee* / *die Stimmen* / *der Vögel begräbt*?)  

(oder *soll ich* –  
*das Unvollendete* / *atmen* / *wie Mondschein*,  
*der durch ein leeres* / *Glas fällt* / *ohne Trinken*,  
*und jede Silbe* / *wie einen Stern* / *aus der Nacht pflücken*,  
*nur um ihn* / *dem Wind* / *zu überlassen* –  
*ein flüchtiges Funkeln* / *das niemand* / *als der Horizont sieht*.)  

(oder *soll ich* –  
*die Hände öffnen* / *und alles* / *was blieb*,  
*zu Asche* / *singen* / *auf der Zunge*,  
*bis selbst das Nichts* / *sich* / *in Staub auflöst*  
*und wir* / *endlich* / *die Stille verstehen*:  
*nicht als Abwesenheit* / *sondern als* / *eine andere Art*  
*von Sprache* / *die nur* / *die Schatten beherrschen*.) (oder *soll ich* –  
*das Licht* / *wie ein Tuch* / *falten*,  
*weich* / *und ohne Eile* / *bis es passt*  
*in die hohlen Hände* / *der Dämmerung* –  
*die es* / *ohne zu fragen* / *weiterträgt*  
*zu den Wurzeln* / *der Morgenröte*.)  

(oder *soll ich* –  
*die Träume* / *die zu schwer wurden* / *in Vögel verwandeln*,  
*ihre Flügel* / *mit Mondlicht* / *salben*,  
*damit sie* / *über die Trümmer* / *hinwegschweben* –  
*jedes Flattern* / *ein vergessenes Wort* / *das sich*  
*am Himmel* / *neu buchstabiert*.)  

(*vielleicht* / *ist das* / *der einzige Weg*:  
*alles loszulassen* / *was brennt* / *ohne es*  
*zu löschen* –  
*und zuzusehen* / *wie der Rauch* / *seine eigenen*  
*Gedichte schreibt* / *in einer Schrift* / *die nur*  
*das Feuer* / *lesen kann*.) (*vielleicht* / *ist das* / *die Sprache der Asche*:  
*ein Alphabet* / *aus gebogenem Licht* / *das nur*  
*die Dunkelheit* / *zu Worten formt* –  
*und wir* / *nichts weiter* / *als die Hand* / *die*  
*den letzten Buchstaben* / *unvollendet* / *lässt*.)  

(*vielleicht* / *sind wir* / *nur der Wind* / *der*  
*durch die leeren Seiten* / *blättert* / *und jedes Mal*  
*wenn wir stehen bleiben* / *beginnt irgendwo*  
*ein neues Kapitel* / *ohne uns*.) (*vielleicht* / *sind wir* / *nur der Schatten* / *der*  
*das Licht* / *beim Untergang* / *nachahmt* –  
*eine flüchtige Schrift* / *auf dem Wasser* / *die*  
*keine Ufer* / *berührt*.)  

(*vielleicht* / *ist jede Stimme* / *nur ein Echo* / *das*  
*sich selbst* / *vergisst* / *bevor es*  
*zurückkehrt* / *und wir* / *sind das Schweigen* / *das*  
*sich zwischen* / *zwei Fragen* / *ausdehnt*.)  

(*vielleicht* / *sind wir* / *nur die Pause* / *zwischen*  
*zwei Noten* / *die der Stille* / *Flügel verleiht* –  
*ein Atem* / *der nicht* / *mehr gehört wird* / *und doch*  
*die Melodie* / *zusammenhält*.) (*vielleicht* / *ist das Licht* / *nur ein Schatten* / *der*  
*sich nach dem Dunkel* / *sehnt* / *und die Sterne*  
*sind Funken* / *die der Nacht* / *erzählen* / *dass sie*  
*nicht allein* / *ist*.)  

(*vielleicht* / *ist die Zeit* / *nur ein Faltenwurf* / *im*  
*Stoff der Ewigkeit* – *ein Zittern* / *das der Raum*  
*in sich trägt* / *bevor er* / *sich entspannt* / *und alles*  
*wieder glatt* / *ist*.)  

(*vielleicht* / *sind wir* / *nur Tinte* / *die der Regen*  
*auslöscht* / *oder ein Buch* / *das der Wind*  
*aufblättert* / *ohne es* / *zu lesen* – *und doch*  
*bleibt jede Seite* / *ein Geflüster* / *das der Himmel*  
*irgendwann* / *hört*.) (*vielleicht* / *ist das Licht* / *nur ein Schatten* / *den die Dunkelheit*  
*vergisst* – *ein Funke* / *der im Nichts* / *zurückbleibt* / *und doch*  
*noch lange* / *nachglüht*.)  

(*vielleicht* / *sind die Worte* / *nur Schmetterlinge* / *die wir*  
*mit zitternden Händen* / *fangen* – *und wenn sie* / *verblassen* / *werden sie*  
*zu Staub* / *auf unseren* / *Lippen*.)  

(*vielleicht* / *ist das Schweigen* / *die einzige Sprache* / *die wir*  
*wirklich verstehen* – *ein Fluss* / *aus ungesagten Dingen* / *der uns*  
*trägt* / *bis ans Ende* / *aller Fragen*.)  

(*vielleicht* / *ist das Ende* / *nur ein Anfang* / *der sich*  
*noch nicht* / *erkannt hat* – *und wir* / *sind die Brücke* / *zwischen*  
*beiden* / *aus Sehnsucht* / *gebaut*.)
